##  Descripción del dataset
Este dataset contiene información de salud de 159 países y 13 columnas, incluyendo esperanza de vida, 
gasto en salud, mortalidad infantil, médicos por cada 1000 habitantes, vacunación y otros indicadores.

##  Cómo ejecutar el notebook
1. Abrir el archivo `taller_modulo1.ipynb`
2. Ejecutar todas las celdas en orden

##  Hallazgos o conclusiones
1. Los países con mayor gasto en salud tienden a tener mayor esperanza de vida.
2. Los países con ingresos bajos presentan mayor mortalidad infantil.
3. Existen diferencias importantes entre regiones en indicadores de salud.
